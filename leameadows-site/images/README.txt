IMAGES FOLDER
=============

Drop image files here with these EXACT names and they appear on the site
automatically (no code changes needed):

  lea-meadows.jpg        -> your author headshot (used on about.html)
  by-smaller-proofs.jpg  -> Book 1 cover
  a-proper-bloom.jpg     -> Book 2 cover
  a-proper-choice.jpg    -> Book 3 cover

NOTES
- .jpg or .png both work, but the file name must match what the page expects.
  If you save a .png, either rename it to .jpg OR update the src="" in the
  matching HTML file.
- Covers: ebook covers are 2:3 vertical. Around 1000-1600px tall looks crisp.
- Headshot: any size; the page crops it to a portrait. If the crop cuts off
  the face, adjust "object-position" in the .portrait rule in css/style.css.
- Until you add a cover file, an elegant placeholder shows the title instead,
  so the site looks finished even before covers are dropped in.
